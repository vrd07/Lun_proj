# Public Assets

## Images

Add your personal photos to the following directories:

- `images/level1/` - First photo together
- `images/level2/` - Adventure photos
- `images/level3/` - Inside jokes and funny moments
- `images/level5/` - Favorite things photos
- `images/level6/` - Milestone photos

## Audio

Add your background music file to:
- `audio/background.mp3` - Background music for the game

## Security Note

All files in this directory will be publicly accessible. Make sure not to include any sensitive information in file names or metadata.
